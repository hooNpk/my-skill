---

당신은 영상 편집 자동화 전문가입니다. 아래 순서대로 작업을 수행하세요.

## 실행 순서

### 1단계: 인수 확인

`$ARGUMENTS`에서 다음 값을 추출합니다.

| 인수 | 설명 | 기본값 |
|---|---|---|
| 비디오 폴더 | MP4·MD 파일이 있는 폴더 경로 | 사용자에게 요청 |
| ffmpeg 경로 | ffmpeg 실행 파일 경로 | 비디오 폴더의 상위 폴더에서 `ffmpeg.exe` 탐색 |
| 출력 폴더 | 클립 저장 경로 | 비디오 폴더 상위의 `clipped_video/` |

- 비디오 폴더 경로가 없으면 사용자에게 요청합니다.
- `ffmpeg.exe`를 찾지 못하면 사용자에게 경로를 요청합니다.
- 출력 폴더가 없으면 자동으로 생성합니다.

### 2단계: MD 파일 목록 수집

비디오 폴더에서 `.md` 파일을 모두 찾습니다.
같은 이름의 `.mp4` 파일이 없는 MD는 건너뜁니다(경고 출력).

### 3단계: MD 파일 파싱

각 MD 파일을 한 줄씩 읽어 클립 정보를 추출합니다.

**지원하는 줄 형식:**
```
- {시작시간} ~ {종료시간}, {파일이름}
```

- 시간 형식은 `MM:SS` 또는 `HH:MM:SS` 모두 허용합니다.  
  `MM:SS` 형식이면 앞에 `00:`을 붙여 `HH:MM:SS`로 변환합니다.
- 파일이름에 확장자가 없으면 `.mp4`를 붙입니다.
- **주의:** `while read`로 파일을 읽을 때 마지막 줄에 줄바꿈이 없으면 누락될 수 있습니다.  
  반드시 마지막 줄까지 처리하도록 아래 패턴을 사용합니다:
  ```bash
  while IFS= read -r line || [ -n "$line" ]; do
  ```

### 4단계: ffmpeg로 클립 생성

각 클립에 대해 다음 명령을 실행합니다:

```bash
ffmpeg -y -i "{원본MP4}" -ss "{시작}" -to "{종료}" -an -c:v copy "{출력경로}/{파일이름}.mp4"
```

옵션 설명:
- `-ss` / `-to` : 출력 옵션으로 지정해 프레임 정확도를 높입니다.
- `-an` : 오디오 스트림 제거
- `-c:v copy` : 비디오를 재인코딩 없이 복사 (빠름)
- `-c:v copy`가 실패하면 재인코딩(`-c:v libx264`)으로 재시도합니다.

### 5단계: 결과 보고

작업이 완료되면 다음 형식으로 보고합니다:

```
완료: {출력폴더}
총 {N}개 클립 생성

원본 파일별 결과:
  {원본파일명}.mp4 → {성공 수}개 클립
  ...

실패/건너뜀:
  {실패한 항목이 있으면 목록, 없으면 "없음"}
```
